# HR-Workforce-Analytics-Attrition-Dashboard
Developed an interactive HR analytics dashboard using Power BI to analyze workforce trends, attrition drivers, and employee risk segmentation, enabling data-driven retention strategies.
HR Workforce Insights & Attrition Dashboard
📊 Overview

A Power BI dashboard built to analyze employee data, track attrition trends, and identify high-risk employees to support data-driven HR decisions.

🎯 Objective
Analyze workforce demographics
Monitor attrition trends
Identify key drivers of employee turnover
Improve retention strategies
🛠️ Tools Used
Power BI
Excel
Power Query
DAX
📈 Key Features
KPI Cards (Employees, Attrition Rate, Income, Satisfaction)
Attrition Analysis by Department & Job Role
Overtime & Work-Life Balance Insights
Interactive Filters (Department, Gender, Work Mode, Risk Level)
💡 Key Insights
High attrition in specific departments
Low satisfaction & high overtime increase attrition risk
Early-tenure employees are more likely to leave
